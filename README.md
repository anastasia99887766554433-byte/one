## AI Daily Digest

Минимальное веб-приложение: показывает главные AI-новости за последние 24 часа и короткую сводку дня.

## Очень коротко (если вы совсем новичок)

1. Откройте **Терминал** (ниже есть инструкция для Windows/macOS/Linux).
2. Перейдите в папку проекта:
   ```bash
   cd /workspace/one
   ```
3. Запустите приложение:
   ```bash
   python3 app.py
   ```
4. Откройте в браузере: `http://localhost:8000`
5. Остановить сервер: нажать `Ctrl + C` в терминале.

---

## Что такое `/workspace/one` и где эта папка?

`/workspace/one` — это **путь к папке проекта в текущей среде** (где сейчас запущен этот проект).

Важно:
- это не «магическая ссылка в интернете», а просто путь на диске/в контейнере;
- у вас на компьютере путь может быть другим (например `C:\Users\Вы\Downloads\one` на Windows или `/Users/you/Downloads/one` на macOS).

Если вы запускаете проект **у себя локально**, используйте путь к той папке, куда вы его скачали/распаковали.

## Как открыть терминал (понятно и по шагам)

### Windows

Способы открыть:
1. Нажмите `Win` и напишите **PowerShell**, откройте его.
2. Или: в папке проекта в Проводнике клик правой кнопкой → **Open in Terminal** (если есть).

Дальше в терминале:
```powershell
cd C:\путь\к\вашей\папке\one
python app.py
```

Если `python` не работает, попробуйте:
```powershell
py app.py
```

### macOS

1. Нажмите `Cmd + Space`, введите **Terminal**, нажмите Enter.
2. В терминале:
```bash
cd /путь/к/вашей/папке/one
python3 app.py
```

### Linux

1. Откройте приложение **Terminal**.
2. Выполните:
```bash
cd /путь/к/вашей/папке/one
python3 app.py
```

После запуска откройте в браузере `http://localhost:8000`.

## Что делать с 5 файлами проекта?

Ничего страшного, вот простое объяснение:

- `app.py` — «движок» приложения (сервер и получение новостей).
- `templates/index.html` — страница сайта.
- `static/styles.css` — внешний вид (цвета, отступы, карточки).
- `README.md` — инструкция.
- `requirements.txt` — список зависимостей (здесь внешние пакеты не требуются).

Для запуска обычно нужен только один шаг: `python3 app.py` (или `python app.py` на Windows).

## Как это работает

- Бэкенд запрашивает Google News RSS по запросу `AI OR artificial intelligence when:1d`.
- Возвращает до 10 новостей с источником и временем публикации.
- Формирует краткую сводку дня на основе часто встречающихся слов в заголовках.
- Если внешний RSS временно недоступен, показывает резервный (fallback) дайджест.

## Если не запускается

- Проверьте Python:
  ```bash
  python3 --version
  ```
  (на Windows можно `python --version`)
- Если видите ошибку про занятый порт `8000`, закройте другой процесс на этом порту и запустите снова.
